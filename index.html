My Projects

Hospital Database Management System

Users from the hospital database can perform CRUD operations. The pharmacist can produce a sales report. A pharmacist can read a prescription that a doctor has added for a specific patient. I have used Java and MySQL database systems. This system does not contain any attractive UI design

Employee Management API

This project provides a REST API with CRUD capabilities for Employees that is built on Spring Boot and MySQL. Spring Security adds Basic Authentication. Additionally, test cases were created.

User Location API

This project involves creating a Spring Boot REST API with CRUD functionality for two user roles, creating and updating a database table using HSQL and JDBC Template, and returning a list of the N nearest users from a given location.

Resgistration-Login-API

Registration-Login-API is a Spring Boot application that provides login and signup functionality using H2 database, Spring Security, Spring Data JPA, and Thymeleaf. It allows users to securely create an account, login, and access a dashboard page.

College Details API

This project is a REST API built with Spring Boot and MySQL, which allows users to retrieve information about colleges and their courses. The API provides two endpoints: one to retrieve a list of all colleges and their associated details, and another to retrieve information about a specific college and its associated courses.
